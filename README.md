# Descripción
FaunaScan Perú surge como respuesta a las dificultades que enfrentan investigadores de fauna y voluntarios ambientales al registrar, identificar y monitorear especies silvestres durante sus actividades de campo. La falta de herramientas accesibles, centralizadas y adaptadas a entornos con conectividad limitada dificulta la recopilación de información confiable para la conservación de la biodiversidad.

La plataforma permite registrar avistamientos de fauna de manera rápida y organizada, incorporando funcionalidades como geolocalización, identificación asistida por inteligencia artificial, almacenamiento de datos en modo offline y generación de reportes para apoyar el monitoreo y la conservación de especies.
# Características
* Registro digital de avistamientos de fauna silvestre.
* Geolocalización de observaciones en tiempo real.
* Identificación de especies mediante inteligencia artificial.
* Acceso y almacenamiento de registros en modo offline.
* Generación de reportes para monitoreo y análisis.
* Interfaz intuitiva para investigadores y voluntarios ambientales.
* Gestión centralizada de información sobre biodiversidad.
* Prototipos diseñados para dispositivos móviles.

* Entrevistas de validación
* Validación de Landing Page
* Validación de prototipos
